### US-09：管理者可以依完成進度區間篩選學員

**作為** 課程管理者
**我想要** 依完成進度區間篩選學員
**以便** 快速查看不同進度區間的學員

**輸入格式**：
- US-06 學員列表
- progressPercent 與區間定義

**輸出格式**：
- 完成進度區間篩選器與更新後列表

**驗收條件**：
- [ ] 提供完成進度篩選器
- [ ] 可篩選：全部、0–25%、26–50%、51–75%、76–100%
- [ ] 篩選後列表即時更新
- [ ] 篩選邏輯正確
- [ ] 手機版可正常操作

**依賴關係**：
- US-06

**優先級**：P0
**相關功能**：Sprint 2｜進度區間篩選

**備註**：
- 讓 Dashboard 更像可操作分析工具，而非靜態圖表。
