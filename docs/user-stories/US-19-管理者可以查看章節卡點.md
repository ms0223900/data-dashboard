### US-19：管理者可以查看章節卡點

**作為** 課程管理者
**我想要** 知道學員最常卡在哪些章節
**以便** 調整教學內容或補充教材

**輸入格式**：
- 章節停留與完成率彙總

**輸出格式**：
- 章節卡點分析區

**驗收條件**：
- [ ] Dashboard 顯示各章節停留人數
- [ ] Dashboard 顯示各章節平均完成率
- [ ] 可以標示停留人數最多的章節
- [ ] 可以顯示章節與作品狀態的關聯，例如卡在 Build Sprint 的學員作品完成情況
- [ ] 數據需與學員列表一致
- [ ] 不需要做到單一影片觀看秒數分析

**依賴關係**：
- US-04、US-06

**優先級**：P2
**相關功能**：Sprint 6｜章節卡點（非本 MVP）
