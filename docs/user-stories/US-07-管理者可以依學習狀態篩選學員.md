### US-07：管理者可以依學習狀態篩選學員

**作為** 課程管理者
**我想要** 依學習狀態篩選學員
**以便** 快速查看學習中、已完課或近期未活動的學員

**輸入格式**：
- US-06 學員列表
- learningStatus 篩選選項

**輸出格式**：
- 學習狀態篩選器
- 篩選後更新的列表；無結果時空狀態

**驗收條件**：
- [ ] 提供學習狀態篩選器
- [ ] 可篩選：全部、尚未開始、學習中、已完成主要章節、已完課、近期未活動
- [ ] 篩選後列表即時更新
- [ ] 篩選後 KPI 或列表數量顯示合理
- [ ] 沒有符合條件時顯示空狀態

**依賴關係**：
- US-06

**優先級**：P0
**相關功能**：Sprint 2｜學習狀態篩選

**備註**：
- 「篩選後 KPI」是否連動待確認（docs/spec.md §15）。影片可展示互動篩選價值。
