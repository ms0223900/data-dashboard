### US-07：管理者可以依學習狀態篩選學員

**作為** 課程管理者
**我想要** 依學習狀態篩選學員
**以便** 快速查看學習中、已完課或近期未活動的學員

**輸入格式**：
- US-06 學員列表
- learningStatus 篩選選項

**輸出格式**：
- 學習狀態篩選器
- 篩選後更新的列表；無結果時空狀態

**驗收條件**：
- [x] 提供學習狀態篩選器
- [x] 可篩選：全部、尚未開始、學習中、已完成主要章節、已完課、近期未活動
- [x] 篩選後列表即時更新
- [x] 篩選後 KPI 或列表數量顯示合理
- [x] 沒有符合條件時顯示空狀態

#### 驗收說明

**整體結論**：PASS ✅

> `/students` 已提供學習狀態篩選（全部＋五狀態）；列表與「目前顯示 N 位」即時更新；無結果顯示空狀態＋清除篩選。KPI／圖表維持全量（Spec §15）。作品／進度篩選見 US-08／US-09。

**依賴關係**：
