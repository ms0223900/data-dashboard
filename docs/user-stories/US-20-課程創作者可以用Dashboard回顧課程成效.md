### US-20：課程創作者可以用 Dashboard 回顧課程成效

**作為** 課程創作者
**我想要** 用 Dashboard 回顧整期課程的學習成效
**以便** 判斷下一期課程要如何調整

**輸入格式**：
- 整期彙總、卡點、作品完成與發布狀況

**輸出格式**：
- 課程復盤摘要／可複製總結文字

**驗收條件**：
- [ ] Dashboard 可以呈現整期課程的完成狀況
- [ ] Dashboard 可以呈現作品完成與發布狀況
- [ ] Dashboard 可以整理出需要優化的章節
- [ ] Dashboard 可以支撐課程復盤討論
- [ ] Dashboard 可以輸出或複製一段課程總結文字
- [ ] Dashboard 可以回扣課程核心流程：Spec、User Stories、Build Sprint、驗收與發布

**依賴關係**：
- US-18、US-19 建議完成

**優先級**：P2
**相關功能**：Sprint 6｜課程復盤（非本 MVP）
