### US-17：助教備註與追蹤

**作為** 助教
**我想要** 對學員加上備註
**以便** 追蹤需要協助的學員

**輸入格式**：
- 備註內容、追蹤標籤
- 持久化儲存（後續）

**輸出格式**：
- 備註區、需要追蹤標示與篩選

**驗收條件**：
- [ ] 助教可以在學員資料中新增備註
- [ ] 備註可以記錄學員卡關點、作品狀態、下次追蹤事項
- [ ] 備註需顯示建立或更新時間
- [ ] 學員列表可以標示「需要追蹤」的學員
- [ ] 可以依「需要追蹤」篩選學員
- [ ] 備註不需要開放給學員查看
- [ ] 第一版不需要多人權限系統，只要先支援管理者／助教內部使用

**依賴關係**：
- US-16

**優先級**：P2
**相關功能**：Sprint 5｜助教協作（非本 MVP）
