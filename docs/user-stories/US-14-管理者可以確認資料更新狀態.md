### US-14：管理者可以確認資料更新狀態

**作為** 課程管理者
**我想要** 知道 Dashboard 的資料最後更新時間
**以便** 判斷目前看到的數據是否可信

**輸入格式**：
- 資料讀取成功／失敗結果
- 最後更新時間戳

**輸出格式**：
- 更新狀態區：已更新／錯誤／過期提醒

**驗收條件**：
- [ ] Dashboard 顯示資料最後更新時間
- [ ] 若資料讀取成功，顯示「已更新」或類似狀態
- [ ] 若資料讀取失敗，顯示錯誤狀態與簡短原因
- [ ] 若資料超過一段時間未更新，顯示提醒
- [ ] 資料更新狀態不影響既有 KPI、圖表與列表的閱讀
- [ ] 手機版仍可清楚看到更新狀態

**依賴關係**：
- US-13

**優先級**：P2
**相關功能**：Sprint 4｜資料同步狀態（非本 MVP）
