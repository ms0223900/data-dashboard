### US-08：管理者可以依作品狀態篩選學員

**作為** 課程管理者
**我想要** 依作品狀態篩選學員
**以便** 快速查看哪些學員正在 Build Sprint、已完成 MVP 或已發布作品

**輸入格式**：
- US-06 學員列表
- projectStatus 篩選選項

**輸出格式**：
- 作品狀態篩選器與更新後列表／空狀態

**驗收條件**：
- [x] 提供作品狀態篩選器
- [x] 可篩選：全部、尚未開始、Build Sprint 中、已完成 MVP、已發布
- [x] 篩選後列表即時更新
- [x] 篩選結果與圖表數據一致
- [x] 沒有符合條件時顯示空狀態

#### 驗收說明

**整體結論**：PASS ✅

> `/students` 作品狀態篩選已上線（與 US-05 同源 `projectStatus`／`PROJECT_STATUS_LABELS`）；可與學習狀態組合；空狀態沿用既有 UI。

**依賴關係**：
- US-06；與 US-05 數據同源

**優先級**：P0
**相關功能**：Sprint 2｜作品狀態篩選

**設計稿來源**：
- Pencil：[`designs/dashboard-screen-3.pen`](../../designs/dashboard-screen-3.pen)
- HTML 參考：[`docs/design/ref/dashboard-screen-3.html`](../design/ref/dashboard-screen-3.html)
- 對應區塊：`篩選列` → `Filter 作品狀態`；示範結果見 `03 學員列表｜已完成 MVP 篩選`
- 實作約束：作品狀態選項與篩選後列表／空狀態互動對齊設計稿；示範情境可對齊「已完成 MVP」篩選 frame。

**備註**：
- 本版本不使用「規劃中」狀態。
