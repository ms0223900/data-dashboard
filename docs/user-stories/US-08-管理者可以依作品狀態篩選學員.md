### US-08：管理者可以依作品狀態篩選學員

**作為** 課程管理者
**我想要** 依作品狀態篩選學員
**以便** 快速查看哪些學員正在 Build Sprint、已完成 MVP 或已發布作品

**輸入格式**：
- US-06 學員列表
- projectStatus 篩選選項

**輸出格式**：
- 作品狀態篩選器與更新後列表／空狀態

**驗收條件**：
- [ ] 提供作品狀態篩選器
- [ ] 可篩選：全部、尚未開始、Build Sprint 中、已完成 MVP、已發布
- [ ] 篩選後列表即時更新
- [ ] 篩選結果與圖表數據一致
- [ ] 沒有符合條件時顯示空狀態

**依賴關係**：
- US-06；與 US-05 數據同源

**優先級**：P0
**相關功能**：Sprint 2｜作品狀態篩選

**備註**：
- 本版本不使用「規劃中」狀態。
