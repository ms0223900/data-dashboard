### US-05：管理者可以看到作品完成狀態分布

**作為** 課程管理者
**我想要** 看到學員作品完成狀態
**以便** 知道學員是否真的把產品做出來

**輸入格式**：
- 假資料 projectStatus 欄位

**輸出格式**：
- 作品完成狀態圖表：人數與比例
- 狀態：尚未開始、Build Sprint 中、已完成 MVP、已發布

**驗收條件**：
- [x] 圖表顯示作品狀態分布
- [x] 狀態包含：尚未開始、Build Sprint 中、已完成 MVP、已發布
- [x] 顯示每個狀態的人數
- [x] 顯示每個狀態的比例
- [x] 數據與學員列表一致

#### 驗收說明

**整體結論**：PASS ✅

> 作品狀態由 `computeProjectStatusSummary(mockStudents)` 彙總；首頁顯示四狀態人數與比例，與同一假資料集一致（學員列表 UI 屬 US-06，資料同源）。
