### US-10：管理者可以看到報表展示區

**作為** 課程管理者
**我想要** 看到一個報表展示區
**以便** 理解這個 Dashboard 如何被用於課程回顧或週報

**輸入格式**：
- 既有 KPI／彙總數字（來自假資料計算）

**輸出格式**：
- 報表展示區：本週重點摘要、關鍵指標摘要
- 適合截圖或錄影；無真實 PDF／CSV 匯出

**驗收條件**：
- [ ] 頁面有報表展示區
- [ ] 報表展示區可以呈現本週重點摘要
- [ ] 報表展示區可以顯示關鍵指標摘要
- [ ] 報表展示區可以用於影片截圖或錄影
- [ ] 不需要真的匯出 PDF 或 CSV

**依賴關係**：
- US-01、US-02、US-03（有可展示數字）

**優先級**：P0
**相關功能**：Sprint 3｜報表展示

**設計稿來源**：
- Pencil：[`designs/dashboard-screen-1.pen`](../../designs/dashboard-screen-1.pen)
- HTML 參考：[`docs/design/ref/dashboard-screen-1.html`](../design/ref/dashboard-screen-1.html)
- 對應區塊：`Bottom Row` → `課程學習成效摘要`
- 實作約束：摘要區標題、關鍵指標文案層級與截圖友善版面以設計稿為準；不做真實 PDF／CSV 匯出。

**備註**：
- 依 Brief，本版本報表匯出展示以截圖或錄影即可。
