### US-04：管理者可以看到章節進度分布

**作為** 課程管理者
**我想要** 看到各章節的完成分布
**以便** 知道學員目前主要完成到哪些階段

**輸入格式**：
- 假資料或 ChapterProgress 彙總
- 固定六個章節名稱

**輸出格式**：
- 章節進度圖表：各章節名稱與完成率或完成人數
- 手機版可閱讀

**驗收條件**：
- [x] 圖表顯示每個章節名稱
- [x] 圖表顯示每個章節完成率或完成人數
- [x] 章節包含：想法收斂、Spec、User Stories、Build Sprint、驗收與修正、發布／延伸應用
- [x] 數據與假資料一致
- [x] 圖表在手機版仍可閱讀

#### 驗收說明

**整體結論**：PASS ✅

> 章節進度由 `computeChapterProgress(mockStudents)` 計算：目前章節序位 ≥ 該章即計入完成；bar＋完成率顯示於首頁圖表區。
