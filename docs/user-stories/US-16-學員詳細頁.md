### US-16：學員詳細頁

**作為** 課程管理者
**我想要** 查看單一學員詳情
**以便** 了解特定學員的進度與作品狀態

**輸入格式**：
- 學員 id／列表列點擊
- 該學員完整欄位

**輸出格式**：
- 詳細頁或詳細側欄

**驗收條件**：
- [ ] 管理者可以從學員列表進入單一學員詳細頁或詳細區塊
- [ ] 詳細頁顯示學員基本資料
- [ ] 詳細頁顯示完成進度與目前章節
- [ ] 詳細頁顯示作品狀態與作品連結
- [ ] 詳細頁顯示最後活動時間
- [ ] 若學員近期未活動，畫面需有明確提示
- [ ] 手機版可以正常閱讀詳細資訊

**依賴關係**：
- US-06

**優先級**：P2
**相關功能**：Sprint 5｜學員詳情（非本 MVP）
