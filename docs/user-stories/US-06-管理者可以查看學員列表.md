### US-06：管理者可以查看學員列表

**作為** 課程管理者
**我想要** 看到學員列表
**以便** 查看每位學員目前的學習進度與作品狀態

**輸入格式**：
- US-02 假資料（建議 ≥20 筆）

**輸出格式**：
- 學員列表表格／區塊，含指定欄位與作品連結（若有）

**驗收條件**：
- [ ] 列表顯示學員姓名
- [ ] 列表顯示 Email
- [ ] 列表顯示完成進度
- [ ] 列表顯示目前章節
- [ ] 列表顯示學習狀態
- [ ] 列表顯示作品狀態
- [ ] 列表顯示最後活動時間
- [ ] 若有作品連結，列表可顯示或開啟作品連結

**依賴關係**：
- US-02

**優先級**：P0
**相關功能**：Sprint 2｜學員列表

**備註**：
- 列表是讓 Dashboard 從「總覽」回到「具體學員」的橋樑。
