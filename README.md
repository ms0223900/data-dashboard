# 數據儀表板 · Data Dashboard

一站式數據儀表板，即時監控關鍵指標與數據可視化。

## 技術棧

- **Framework**：Next.js 16 (App Router)
- **Language**：TypeScript
- **Styling**：Tailwind CSS 4
- **Database**：Supabase (PostgreSQL)
- **Deploy**：Vercel

## 環境需求

- [Node.js](https://nodejs.org/) **18+**（建議 20 或 22 LTS）
- [npm](https://www.npmjs.com/)（隨 Node 安裝）

## 安裝

```bash
# 1. 進入專案目錄
cd data-dashboard

# 2. 安裝依賴
npm install

# 3. 複製環境變數範本
cp .env.example .env.local
```

## 開發

```bash
npm run dev
```

打開 [http://localhost:3000](http://localhost:3000) 檢視結果。
